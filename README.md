# Market Data Support & Analytics Project

## Overview
This project simulates a market data support environment using PostgreSQL and Tableau with SPY ETF historical data. It focuses on data ingestion, validation, and financial time-series analysis.

## Tools Used
- PostgreSQL
- Tableau
- SQL (DBeaver)

## Dataset
SPY ETF historical market data (Kaggle)

## Key Features

### Data Ingestion
- Loaded CSV market data into PostgreSQL
- Mapped fields to relational schema

### Data Quality Checks
- OHLC integrity validation
- Missing data detection
- Volume anomaly checks

### Financial Analysis
- Daily returns calculation
- Volatility analysis
- Trend analysis over time

## Tableau Dashboard
Includes:
- Price trend visualisation
- Returns analysis
- Data quality KPI monitoring (OHLC, Missing Data, Volume)

## Key Skills Demonstrated
- SQL querying and transformation
- Data validation and quality control
- Time-series financial analysis
- Dashboard development in Tableau
